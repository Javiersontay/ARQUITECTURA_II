#Explicación del uso de Husky
Husky fue configurado para ejecutar validaciones automáticas antes de realizar un commit.

## Lo que hace Husky en este proyecto es lo siguiente:
- Ejecuta `eslint` antes de permitir el commit.
- Previene subir código con errores de linting.
- Asegura mayor calidad del código antes de enviarlo al repositorio.
